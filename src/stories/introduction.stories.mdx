# Cosmic UI

Cosmic UI is a set of commonly used components built upon TailwindCSS created to facilate high degrees of customization in a simple to modify fashion.

## **Color Language**

Coismic uses a color variant system in where each variant has it's own set of colors associated with it.

The basic variants are:

-   **primary**
-   **secondary**
-   **error**
-   **warning**
-   **success**
-   **ghost**
-   **surface**

Each of these **variants** have a subset of shades for different states such as `hover` and `disabled`,

Please see the Color Language Page for more details.

## **TailwindCSS Usage**

While you are not required to use TailwindCSS in your projects, our library uses TailwindCSS under the hood and be easily extended using the `tailwind.config.js` file. Every component is designed to be able to take in additional tailwind classes to further customize your layouts.

### **Tailwind Configuration and Principles**

If you decide to use TailwindCSS in alongside this project, you will be able to control the whole theme from the core `tailwind.config.js` file. If you know how to set custom values in this file, you know how to customize the base theme!

To get started you will need to add the `CosmicPlugin` to the tailwind config file.

```javascript
module.exports = {
    ...,
    plugins: [CosmicPlugin()]
}
```

You may overwrite anything in the cosmic plugin by simply naming the key the same as a cosmic key in the configuration. For example, the default `primary` color variant is a blue color `#3b82f6` and if I wanted to change it to red I would do the following.

```javascript
module.exports = {
    ...,
    theme: {
        primary: { DEFAULT: '#FF0000' } // #FF0000 is red
    }
}
```

**Note:** Cosmic will merge any value you provide in this file, with our default values. User defined values always take precedence. This means if you were to do the above you will only overwrite the `primary` variant and the base theme `secondary` `success` `error` and other variants will remain untouched.

### Tailwind Base theme

The base theme TailwindCSS Configuration looks like this.

```javascript
module.exports = {
    theme: {
        extend: {
            borderRadius: {
                'theme-button': '0.25rem',
                'theme-input': '0.25rem'
            },
            fontSize: {
                'theme-input-description': '0.7rem',
                'theme-input-label': '0.8rem'
            },
            transitionProperty: {
                width: 'width'
            },
            animation: {
                indeterminate: 'indeterminate 1s linear infinite'
            },
            keyframes: {
                indeterminate: {
                    '0%': { transform: 'translateX(0) scaleX(0)' },
                    '40%': { transform: 'translateX(0) scaleX(0.4)' },
                    '100%': { transform: 'translateX(100%) scaleX(0.5)' }
                }
            },
            ringWidth: {
                'theme-radio-border-width': '5px'
            }
        },
        colors: {
            primary: { DEFAULT: '#3b82f6', pair: '#fafbfe', hover: '#0065FF', disabled: '#4C9AFF' },
            secondary: { DEFAULT: '#172B4D', pair: '#fafbfe', hover: '#344563', disabled: '#42526E' },
            purple: { DEFAULT: '#6554c0', pair: '#fafbfe', hover: '#8777D9', disabled: '#998DD9' },
            ghost: { DEFAULT: '#ebecf0', pair: '#172B4D', hover: '#DFE1E6', disabled: '#F4F5F7' },
            error: { DEFAULT: '#fe5531', pair: '#1c2d4c', hover: '#FF7452', disabled: '#FF8F73' },
            warning: { DEFAULT: '#FF991F', pair: '#1c2d4c', hover: '#FFC400', disabled: '#FFE380' },
            success: { DEFAULT: '#7af1c0', pair: '#016946', hover: '#57D9A3', disabled: '#ABF5D1' },
            surface: { DEFAULT: '#F4F5F7', pair: '', hover: '#ebecf0', disabled: '' },
            heading: { DEFAULT: '#172B4D', pair: '', hover: '', disabled: '' },
            body: { DEFAULT: '#6B778C', pair: '', hover: '', disabled: '' },
            white: { DEFAULT: '#ffffff', pair: '', hover: '', disabled: '' },
            black: { DEFAULT: '#000000', pair: '', hover: '', disabled: '' }
        }
    }
};
```
