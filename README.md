# Cosmic UI

## Installation

```bash
yarn add @aspiralabs/cosmic

-- OR --

npm i @aspiralabs/cosmicui
```

Then add Cosmic UI to your `tailwind.config.js`

```javascript
const CosmicPlugin = require('@aspiralabs/cosmicui/dist/src/plugin');

module.exports = {
    content: ['*'],
    plugins: [CosmicPlugin]
};
```
